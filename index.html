Portfolio

The projects below, were created during the intensive three month full-time web-development program I completed at General Assembly. The course is designed to provide a core foundation to become a full stack junior developer as the units covered core fundamentals of front end and back end development.

Project 1: Guess That Movie!

Build: JavaScript ES6, jQuery 3.10, HTML5, SCSS

Build: JavaScript ES6, jQuery 3.10, HTML5, SCSS.

My first project involved building a game using JavaScript. The premise of my game is based on a user guessing nine movie titles, within a time limit. The user is given a movie title in the form of jumbled tiles/scrabble styled letters, an artistic poster, a theme song and a hint/quote from that movie. The user has to click on each letter tile to spell out the full title. You can return the letter back to its original position by simply clicking on it again.

Project 2: Event Hub

New tech: Node.js, Express.js, Bcrypt, MongoDB, AJAX, Skiddle API

New tech: Node.js, Express.js, Bcrypt, MongoDB, AJAX, Skiddle API.

My first RESTful app. I built an events app where the user can view a list of events in various cities in England. I also included a radius slider should the user want to be more specific. The user can save an event that they are interested in to their ‘MyEvents’ section on their profile page. I used Skiddle’s API to incorporate their events data into my app. Using the MTV News’ API, I added a news feed which displays trending articles from MTV News. I also used Bcrypt to hash users’ passwords.
